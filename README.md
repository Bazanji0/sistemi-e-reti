# Sistemi e Reti — Studio Maturità

Applicazione full-stack per studiare **Sistemi e Reti** per la maturità italiana.

## Tech Stack

- **Frontend**: React 18 + Vite + Tailwind CSS + React Router
- **Backend**: Node.js + Express + better-sqlite3
- **Grafici**: Recharts
- **Mappa concettuale**: React Flow

## Avvio rapido

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (in un altro terminale)
cd frontend
npm install
npm run dev
```

Apri http://localhost:5173

## Funzionalità

- **Dashboard** — Panoramica progresso per sezione con statistiche globali
- **Studio** — Navigazione per sezione, contenuti markdown, segna come studiato, bookmarks
- **Quiz** — Risposta multipla per sezione o misto, timer opzionale, storico con grafici
- **Flashcards** — Flip animation, spaced repetition, filtro per sezione
- **Simulazione orale** — Domande aperte stile maturità con risposte modello
- **Mappa concettuale** — Visualizzazione interattiva connessioni tra argomenti
- **Glossario** — Lista alfabetica termini tecnici con ricerca
- **Ricerca globale** — Cerca in argomenti, quiz e glossario

## Contenuti

14 sezioni complete: Mezzi trasmissivi, Apparati di rete, Cablaggio strutturato, Ethernet, Modelli OSI/TCP-IP, Indirizzi IP, Subnetting FLSM, VLSM, Routing, Routing statico, Grafi, RIP/OSPF, Trasporto, Applicazioni e QUIC.
